## 技术要点

### CSS类使用规范
- 使用现有的 `.content-section` 作为页面容器
- 使用 `.section-header` 作为页面标题区域
- 使用 `.card` 或 `.info-card` 作为内容卡片
- 使用 `.grid` 或 `.flex` 布局类

### 图标使用
- 使用 Font Awesome 图标库
- 主菜单图标：fas fa-chart-line, fas fa-compass 等
- 子菜单图标：fas fa-star, fas fa-child 等

### JavaScript交互
- 页面切换使用现有的 showSection() 函数
- AI助手使用 toggleChatPopup() 函数
- 确保所有点击事件正确绑定
